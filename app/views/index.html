Iteration 4

07 Feb 2019 - 20 Feb 2019

  • Apply v1.2 - apply for you, successful application

Start page

  • Added in ‘the age of’ to content for users where English is not their first language
  • Removed messaging around receiving an instant decision to align with current dev API access.

Who is applying

  • Simplified ‘I am pregnant, or am a parent or guardian with one or more children under the age of 4’ - by splitting it into 2 questions rather than 1. Those where English is not their 1st language found this difficult to understand
  • Changed language from ‘on behalf of’ to ‘for’ as it confused some users

Which of the following best describes you

  • This is a new question as described in point 2 of 'who is applying'.
  • 'Under 4' caused confusion in V1 testing as some users questioned if a 4-year old was still eligible. We are trying 'aged 3 or under' instead.
  • Some users may be pregnant and have children, so we are testing a combined option for clarity.

When is your baby due to be born

  • This is now a conditional question if the user selects ‘I am pregnant’ which explicitly asks for the baby’s due date from ‘Which of the following describes you’. 1 user was confused by being asked this question after selecting a statement which already said she was pregnant on the first question.

Delivery address

  • The H1 has been changed to make it clearer that it is a delivery address we are asking for. All users missed this.
  • Address labels have been changed as the word 'building' caused some confusion, as she wasn't sure if this meant the house number, in the end she left off the house number, as it wasn't actually specifically asking for that.

Confirmation page

  • The negative statements from the confirmation page such as ‘you have told us you are not pregnant’ etc have been removed. Users asked why we were showing them this information and felt it was unnecessar.
  • The H1 has been changed to emphasise this is the end of the application process.
  • Numbered next steps have been added to increase user engagement. Few users read this page in V1 testing.

Privacy notice

  • Added and linked to from the start and check your answers pages.
  • Illustrative data retention added until true facts are known.
  • Consent added to the declaration on the check your answers screen following GDPR advice

Cookies

  • Added and linked to from the privacy notice page.
